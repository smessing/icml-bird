# Environment:

## Bash Variables
* TRAIN\_DATA\_MFCC\_PATH should be defined as the absolute path to the training
  data's MFCC location. E.g.,

    export TRAIN\_DATA\_MFCC\_PATH="/Users/sam/icml-bird/data/training/mfcc"
